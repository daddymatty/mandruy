# Мандруй — сервісний портал 24/7 для подорожей

Статичний двомовний (UK/EN) сайт-портал послуг для подорожей: **страхування, ВНЖ і документи, квитки, тури, готелі, eSIM і трансфери**. Зроблено на **чистому HTML5, CSS3 та JavaScript** — без конструкторів, CMS і зовнішніх залежностей у рантаймі. HTML генерується власним статичним генератором зі структурованого контенту (data-driven).

## ✨ Ключові характеристики

- **Максимально анімований, але простий** дизайн: плавні появи при скролі, анімовані градієнти, «орби», маркі, лічильники, живі мікро-взаємодії. Усе на `transform`/`opacity` і повністю вимикається при `prefers-reduced-motion`.
- **Двомовність (UK/EN)** з коробки: українська — у корені `/`, англійська — у `/en/`, коректні `hreflang` та перемикач мов.
- **Швидкість**: критичний CSS інлайниться, JS із `defer`, inline-SVG (нуль зайвих запитів), без зовнішніх шрифтів і бібліотек.
- **Технічне SEO**: semantic HTML, Open Graph, Twitter Cards, JSON-LD (TravelAgency, WebSite, Service, Article, FAQPage, BreadcrumbList), canonical, XML Sitemap з альтернативами, robots.txt.
- **Конверсія**: форми-ліди на кожній сторінці, sticky-CTA для мобільних, плаваючі Telegram/WhatsApp, кнопка дзвінка.

## 📁 Структура

```
mandruy/
├── build.js                 # генератор: дані + шаблони → dist/
├── src/
│   ├── data/                # контент як ES-модулі (bilingual)
│   │   ├── site.js          # конфіг, контакти, локалі, навігація
│   │   ├── services.js      # 7 послуг (тарифи, переваги, FAQ)
│   │   ├── blog.js          # статті блогу
│   │   └── faq.js           # FAQ головної
│   ├── lib/
│   │   ├── i18n.js          # словник UI + хелпери локалізації
│   │   └── seo.js           # генератори JSON-LD Schema.org
│   ├── templates/           # HTML-шаблони (функції → рядок)
│   └── assets/              # css, js, svg (лого, favicon, OG)
├── tools/serve.js           # локальний сервер для перегляду
└── dist/                    # згенерований сайт (для деплою)
```

## 🚀 Швидкий старт

```bash
npm run build      # згенерувати сайт у dist/
npm run serve      # підняти локальний сервер на http://localhost:8080
npm run dev        # build + serve
```

Потрібен лише **Node.js ≥ 18**. Dev-залежності (`csso`, `terser`, `html-minifier-terser`) опційні — потрібні лише для мініфікації; без них збірка теж працює.

## 🌐 Деплой

`dist/` — статика, яку можна розмістити будь-де: Cloudflare Pages, Netlify, GitHub Pages, Nginx, S3+CloudFront. Перед продакшеном:

1. Замініть домен і контакти у `src/data/site.js` (`domain`, телефон, e-mail, месенджери, соцмережі).
2. Підключіть форми до бекенду: вкажіть `leadEndpoint` (проксі до Telegram/CRM) — інакше заявки зберігаються лише в `localStorage`.
3. За потреби додайте GA4 (`analytics.ga4`) і верифікацію Search Console.

## 🧩 Як додати контент

- **Нова послуга** — додайте об’єкт у `src/data/services.js` (з полями `uk`/`en`).
- **Нова стаття** — додайте об’єкт у `src/data/blog.js`.
- **Новий рядок інтерфейсу** — додайте ключ у `src/lib/i18n.js` для обох мов.

Після змін запустіть `npm run build`.
